## Logo

### Description
The Home Depot logo features the company name in bold Stencil D font with the distinctive orange (`#F96302`) and white color scheme. The logo has remained visually consistent since 1978.

### Files
- `logo.svg` - Vector logo

### Usage Guidelines
- Maintain clear space around logo
- Do not alter colors, proportions, or font
- Use on white or light backgrounds for best visibility
- Orange background with white logo is acceptable

## Visual Design System

### Brand Aesthetic
Home Depot's visual identity draws from its **industrial warehouse heritage** — bold, functional, and approachable. The design language communicates strength, reliability, and accessibility.

**Core Design Principles:**
| Principle        | Description                                                    |
|------------------|----------------------------------------------------------------|
| **Bold & Direct**| High-contrast, no-frills design that commands attention        |
| **Industrial**   | Stencil typography, warehouse-inspired, utilitarian feel       |
| **Clean**        | Minimalist layouts with strong visual hierarchy                |
| **Approachable** | Friendly and empowering, not intimidating                      |
| **Trustworthy**  | Professional, reliable, consistent                             |

### Document Styling

**Page Layout:**
- Use generous white space — don't overcrowd
- Align content to a clear grid system
- Orange used sparingly as accent, not dominant
- White or light gray (`#F5F5F5`) backgrounds preferred

**Headers & Titles:**
- Headlines: Helvetica Neue LT Pro 77 Bold Condensed, Black (`#000000`)
- Use ALL CAPS for major section headers
- Orange underline/accent bar beneath headers (2-4px thick)

**Body Content:**
- Body text: Helvetica Neue LT Pro 55 Roman, Dark Gray (`#747474`) or Black
- Line height: 1.4–1.6 for readability
- Paragraph spacing: 12–16px

**Accent Elements:**
- Orange (`#F96302`) for: CTAs, highlights, underlines, icons
- Warm Gray Medium (`#C4C4C4`) for: dividers, borders, secondary elements
- Warm Gray Light (`#F5F5F5`) for: backgrounds, cards, containers

### Mockup & Presentation Guidelines

**Slide/Document Structure:**
```
┌─────────────────────────────────────────────┐
│  [LOGO - top left]              [Page #]   │
├─────────────────────────────────────────────┤
│                                             │
│  SECTION HEADER                             │
│  ───────────── (orange accent line)         │
│                                             │
│  Body content here with clear hierarchy     │
│  • Bullet points use orange dots            │
│  • Keep text concise and scannable          │
│                                             │
│                                             │
├─────────────────────────────────────────────┤
│  Footer: Source / Date / Confidentiality    │
└─────────────────────────────────────────────┘
```

**Color Usage Ratios:**
| Color          | Usage                              | Approximate % |
|----------------|------------------------------------|--------------:|
| White          | Primary background                 | 60-70%        |
| Black          | Headlines, key text                | 15-20%        |
| Gray (all)     | Body text, secondary elements      | 10-15%        |
| Orange         | Accents, CTAs, highlights only     | 5-10%         |

**Photography Style:**
- Real people (DIYers, pros, families) — authentic, not staged
- In-store or project environments
- Warm, well-lit imagery
- Action-oriented (people doing, building, creating)

### Graphic Elements

**Orange Accent Bar:**
- Used beneath headers or as section dividers
- Thickness: 2–4px for digital, 3–6pt for print
- Full width or partial (left-aligned)

**Box/Container Motif:**
- Square/rectangular shapes represent "all-inclusive solution"
- Use rounded corners sparingly (4px max) or sharp corners
- Orange borders on white cards for emphasis

**Icons:**
- Simple, bold line icons preferred
- Stroke weight: 2px
- Orange or Black
- Consistent style throughout

### Do's and Don'ts

| Do ✓ | Don't ✗ |
|------|---------|
| Use orange as accent only | Don't use orange as background for large areas |
| Keep layouts clean and spacious | Don't overcrowd with text or elements |
| Use bold, condensed headlines | Don't use decorative or script fonts |
| Maintain strong contrast | Don't use low-contrast color combinations |
| Use official colors only | Don't introduce new colors to the palette |
| Keep the industrial, direct feel | Don't make designs feel "soft" or overly polished |

## Brand Voice & Tone

### Tagline
**"Let's Do This"**

### Personality
- Knowledgeable and trustworthy
- Grounded and confident with warmth
- Empowering DIY can-do attitude

### Communication Style
- **Informative** — Expert knowledge, helpful guidance
- **Friendly** — Approachable, never condescending
- **Direct** — Clear, action-oriented language
- **Empowering** — "You can do it. We can help."
